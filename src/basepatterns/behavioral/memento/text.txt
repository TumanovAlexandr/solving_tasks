Шаблон проектирования "Хранитель" (Memento pattern) - это поведенческий шаблон проектирования, который позволяет
сохранять и восстанавливать состояние объекта без нарушения инкапсуляции.

В шаблоне проектирования Хранитель создается объект-хранитель, который содержит состояние объекта, которое нужно
сохранить. Затем создается объект-создатель, который использует объект-хранитель для сохранения или восстановления
состояния объекта. Объект-создатель может создавать несколько объектов-хранителей, чтобы сохранить несколько состояний объекта.

Основные составляющие шаблона проектирования Хранитель:

Хранитель (Memento) - это класс, который содержит состояние объекта, которое нужно сохранить.
Создатель (Originator) - это класс, который создает объект-хранитель для сохранения состояния объекта или использует
объект-хранитель для восстановления состояния объекта.
Опекун (Caretaker) - это класс, который хранит объект-хранитель и предоставляет методы для доступа к объекту-хранителю.
