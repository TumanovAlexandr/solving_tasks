Шаблон проектирования "Состояние" (State pattern) - это поведенческий шаблон проектирования, который позволяет объекту
изменять свое поведение в зависимости от своего внутреннего состояния. Шаблон Состояние предполагает, что у объекта может
быть несколько состояний, каждое из которых определяет его поведение в определенный момент времени.

Основные составляющие шаблона проектирования Состояние:

Контекст (Context) - это класс, который содержит объект состояния и предоставляет интерфейс для взаимодействия с объектом
состояния.
Состояние (State) - это интерфейс или абстрактный класс, который определяет методы для обработки различных действий.
Конкретные состояния (Concrete States) - это классы, которые реализуют интерфейс или наследуют абстрактный класс
состояния и предоставляют конкретную реализацию методов для обработки действий.
