Шаблон проектирования "Итератор" (Iterator pattern) - это поведенческий шаблон проектирования, который позволяет
последовательно обходить элементы коллекции, не раскрывая ее внутреннюю структуру.

В шаблоне проектирования Итератор создается класс-итератор, который содержит методы для обхода элементов коллекции.
Затем коллекция предоставляет метод, который возвращает объект-итератор, который можно использовать для последовательного
обхода элементов коллекции.

Основные составляющие шаблона проектирования Итератор:

Итератор (Iterator) - это интерфейс или абстрактный класс, который определяет методы для обхода элементов коллекции.
Конкретный итератор (Concrete Iterator) - это класс, который реализует интерфейс Iterator и предоставляет реализацию для
обхода элементов коллекции.
Коллекция (Collection) - это интерфейс или абстрактный класс, который определяет метод для создания объекта-итератора.
Конкретная коллекция (Concrete Collection) - это класс, который реализует интерфейс Collection и предоставляет реализацию
для создания объекта-итератора.
