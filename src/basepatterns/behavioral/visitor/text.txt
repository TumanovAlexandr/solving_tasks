Шаблон проектирования "Посетитель" (Visitor pattern) - это поведенческий шаблон проектирования, который позволяет
добавлять новые операции к объектам, не изменяя классы объектов. При использовании шаблона проектирования Посетитель,
создается отдельный класс, который содержит методы для обработки различных типов объектов. Затем этот класс передается в
качестве параметра к объектам, которые нужно обработать.

Основные составляющие шаблона проектирования Посетитель:

Посетитель (Visitor) - это класс, который содержит методы для обработки различных типов объектов.
Элемент (Element) - это интерфейс, который определяет метод accept(), принимающий в качестве параметра объект посетителя.
Конкретный элемент (Concrete Element) - это класс, который реализует интерфейс Element и реализует метод accept(),
передавая себя в качестве параметра к методам посетителя.
Объект структуры (Object Structure) - это класс, который содержит коллекцию объектов Element.
