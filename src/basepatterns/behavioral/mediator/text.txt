Шаблон проектирования "Посредник" (Mediator pattern) - это поведенческий шаблон проектирования, который позволяет
уменьшить связность между объектами, позволяя им взаимодействовать друг с другом через центральный объект-посредник.

В шаблоне проектирования Посредник создается класс-посредник, который содержит методы для координации взаимодействия
между объектами. Объекты, которые должны взаимодействовать друг с другом, сохраняют ссылку на объект-посредник. Когда
объект нуждается в выполнении какой-либо операции, он вызывает метод на объекте-посреднике, который затем координирует выполнение операции между соответствующими объектами.

Основные составляющие шаблона проектирования Посредник:

Посредник (Mediator) - это интерфейс или абстрактный класс, который определяет методы для координации взаимодействия
между объектами.
Конкретный посредник (Concrete Mediator) - это класс, который реализует интерфейс Mediator и содержит конкретную
реализацию для координации взаимодействия между объектами.
Коллега (Colleague) - это класс, который содержит ссылку на объект-посредник и может вызывать методы на объекте-посреднике
для выполнения операции.
