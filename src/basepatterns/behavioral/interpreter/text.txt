Шаблон проектирования "Интерпретатор" (Interpreter pattern) - это поведенческий шаблон проектирования, который позволяет
описывать грамматику языка и интерпретировать выражения на этом языке.

В шаблоне проектирования Интерпретатор создается грамматика языка и набор классов, которые представляют различные
элементы грамматики. Затем создается интерпретатор, который может интерпретировать выражения, представленные объектами-элементами грамматики.

Основные составляющие шаблона проектирования Интерпретатор:

Абстрактное выражение (Abstract Expression) - это интерфейс или абстрактный класс, который определяет метод interpret()
для интерпретации выражения.
Терминальное выражение (Terminal Expression) - это класс, который реализует интерфейс Abstract Expression и предоставляет
реализацию для интерпретации терминальных выражений.
Нетерминальное выражение (Nonterminal Expression) - это класс, который реализует интерфейс Abstract Expression и
предоставляет реализацию для интерпретации нетерминальных выражений.
Контекст (Context) - это класс, который содержит информацию, необходимую для интерпретации выражений.
Клиент (Client) - это класс, который создает объекты-элементы грамматики и контекст и вызывает метод interpret() на
корневом объекте-элементе грамматики.
