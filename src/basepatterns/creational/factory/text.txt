Шаблон проектирования "Фабрика" (Factory pattern) - это порождающий шаблон проектирования, который позволяет создавать
объекты без явного указания их класса.

В шаблоне проектирования Фабрика используется метод фабрики, который создает объекты, не раскрывая свою реализацию.
Метод фабрики обычно реализуется в виде статического метода, который возвращает объекты определенного типа.

Основные составляющие шаблона проектирования Фабрика:

Продукт (Product) - это интерфейс, который определяет общие методы для всех продуктов.
Конкретный продукт (Concrete Product) - это класс, который реализует интерфейс Product и представляет конкретный тип продукта.
Фабрика (Factory) - это интерфейс или абстрактный класс, который определяет методы для создания объектов типа Product.
Конкретная фабрика (Concrete Factory) - это класс, который реализует интерфейс Factory и создает конкретные объекты типа Product.
