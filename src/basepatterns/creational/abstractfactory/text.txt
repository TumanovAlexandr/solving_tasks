Шаблон проектирования "Абстрактная фабрика" (Abstract Factory pattern) - это порождающий шаблон проектирования, который
позволяет создавать семейства связанных объектов без указания их конкретных классов.

В шаблоне проектирования Абстрактная фабрика создается абстрактная фабрика, которая определяет методы для создания
связанных объектов. Конкретные фабрики реализуют абстрактную фабрику и создают конкретные объекты, которые соответствуют определенному семейству.

Основные составляющие шаблона проектирования Абстрактная фабрика:

Абстрактная фабрика (Abstract Factory) - это интерфейс или абстрактный класс, который определяет методы для создания
связанных объектов.
Конкретная фабрика (Concrete Factory) - это класс, который реализует интерфейс Abstract Factory и создает конкретные
объекты, которые соответствуют определенному семейству.
Абстрактный продукт (Abstract Product) - это интерфейс или абстрактный класс, который определяет общие методы для всех
продуктов в семействе.
Конкретный продукт (Concrete Product) - это класс, который реализует интерфейс Abstract Product и представляет
конкретный тип продукта в семействе.
