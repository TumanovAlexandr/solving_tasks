Шаблон проектирования "Строитель" (Builder pattern) - это порождающий шаблон проектирования, который позволяет создавать
сложные объекты пошагово, разделяя процесс их создания на отдельные этапы.

В шаблоне проектирования Строитель создается отдельный класс-строитель, который предоставляет методы для пошагового
создания объекта. Каждый метод добавляет новую часть объекта, пока объект не будет полностью создан. Класс-строитель может
использоваться для создания различных вариантов объекта, используя один и тот же процесс строительства.

Основные составляющие шаблона проектирования Строитель:

Продукт (Product) - это класс, который представляет создаваемый объект.
Строитель (Builder) - это интерфейс, который определяет методы для пошагового создания объекта.
Конкретный строитель (Concrete Builder) - это класс, который реализует интерфейс Builder и предоставляет методы для
пошагового создания конкретной реализации объекта.
Директор (Director) - это класс, который использует объекты типа Builder для создания объекта типа Product.
