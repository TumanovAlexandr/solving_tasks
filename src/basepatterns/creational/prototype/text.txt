Шаблон проектирования "Прототип" (Prototype pattern) - это порождающий шаблон проектирования, который позволяет
создавать новые объекты путем клонирования существующего объекта, вместо создания нового объекта с нуля.

В шаблоне проектирования Прототип создается объект-прототип, который является примером для создания новых объектов.
Затем используется метод клонирования, который позволяет создавать новые объекты на основе существующего объекта-прототипа.
Клонирование может быть глубоким или поверхностным, в зависимости от того, какую часть объекта нужно скопировать.

Основные составляющие шаблона проектирования Прототип:

Прототип (Prototype) - это интерфейс или абстрактный класс, который определяет метод клонирования.
Конкретный прототип (Concrete Prototype) - это класс, который реализует интерфейс Prototype и предоставляет методы для
клонирования существующего объекта.
