Шаблон проектирования фасад (Facade pattern) - это структурный шаблон проектирования, который позволяет скрыть сложность
системы, предоставляя упрощенный интерфейс для взаимодействия с ней.

В шаблоне проектирования фасад создается класс-фасад, который предоставляет упрощенный интерфейс для клиента, скрывая
сложность системы. Класс-фасад делегирует вызовы методов в соответствующие объекты внутри системы, обеспечивая связь
между клиентом и внутренними объектами системы.

Основные составляющие шаблона проектирования фасад:

Фасад (Facade) - это класс, который предоставляет упрощенный интерфейс для взаимодействия с системой.
Система (Subsystem) - это группа классов, которые реализуют функциональность системы.
